# Launch Checklist

## Streamlit

1. Run `streamlit run app.py`
2. Create the local passcode
3. Complete onboarding
4. Save one quiz result
5. Export CSVs and markdown summary
6. Verify home-lab lock/unlock flow

## Public Link

1. Choose Streamlit Community Cloud or Cloudflare Tunnel
2. Generate the public app URL
3. Save the public URL in `Access And Share Settings`

## Mobile

1. `cd mobile_app`
2. `npm install`
3. `npm run start`
4. `npx eas build --platform all --profile preview`
5. Save the mobile URL in `Access And Share Settings`

## Portfolio

1. Add app screenshots to the repo
2. Commit `DEPLOYMENT.md`
3. Commit the latest `data/exports/study_summary.md` if you want a public study log
4. Add the public links to the GitHub README and LinkedIn post
